## 小结

本类药物以外用为主，具有杀虫、解毒、消肿止痛、化腐生肌、收敛止血等功效。主要用于痈疽疮疖、疥癣、虫蛇咬伤及耳目喉鼻等疾患。应用方法有：膏贴、涂擦、吹喉、滴鼻、点眼、薰洗、熨烫等。因大多外用药具有不同程度的毒性，所以，应慎重使用，防止中毒。

硫黄、雄黄为解毒杀虫药，用于疥癣恶疮。二药相比，硫黄止痒杀虫效佳，主治疥癣；雄黄解毒疗疮功强，主治痈疽恶疮。雄黄经配伍制成丸、散内服，用于虫积腹痛、疟疾、痢疾及痰涎壅盛之症，硫黄经配伍制成丸、散内服，用于阳虚之痰喘、阳痿及虚寒便秘。

砒石、水银均有剧毒，为解毒杀虫药，主治疥癣恶疮。二药相比，砒石兼有蚀疮去腐作用，为治恶疮腐肉不去要药；水银无此功效。二药相反，不可同用。砒石经配伍制成丸、散内服，可治疟疾、痰喘；水银今时已不用于内服。轻粉性味、功效与水银相似，轻粉内服有缓泻、利尿作用。

铅甘寒入肾，安神明目，坠痰杀虫。铅丹，又名黄丹，辛微寒，入心肝经、有解毒、止痒、生肌作用。可配制外贴膏药，用于疮痈肿毒久溃不敛；铅粉，又名宫粉、胡粉、锡粉，辛寒有毒，为外用药，与铅丹功效相似。密陀僧与铅丹、铅粉的作用相似。

硼砂、炉甘石解毒防腐，为眼科常用。二药相比，硼砂性凉，清热解毒，清肺化痰，外用主治口舌生疮、咽喉肿痛、目赤翳膜，内服主治肺热痰壅；炉甘石性平，收湿敛疮，专供外用，主治湿疮、湿疹多脓水而不收口者。

明矾、皂矾功效相似，均有解毒杀虫、燥湿化痰作用，外用主治疮疡湿疹、疥疮、口舌生疮、耳中流脓。明矾内服止血止泻，用于外伤出血、便血、崩漏及久泻不止诸症；皂矾内服补血消积，用于肠寄生虫病引起的贫血、萎黄病。

斑蝥、蟾蜍辛而有大毒，对皮肤有刺激作用。二药相比，斑蝥性寒，外用攻毒蚀疮，治痈疽、瘰疬、顽癣，内服破血散结，治经闭、癥瘕、狂犬咬伤；蟾蜍性温，外用攻毒消肿止痛，治痈肿疔毒、咽喉肿痛，内服开窍辟秽，治吐泻腹痛、神志昏迷。

木鳖子通络散结，消肿止痛，治痈疽肿毒，跌打损伤，风湿痹痛，拘挛麻木等症。本药有小毒，以外用为主，极少内用，严格控制用量。

血竭外用止血生肌敛疮，治溃疡不敛，外伤出血，内服活血散瘀止痛，治跌打损伤，瘀血肿痛。

樟脑外用除湿杀虫、温散止痛，治疥癣、牙痛、跌打损伤；内服芳香开窍、辟秽化浊、温散止痛，治神志昏迷、吐泻腹痛。

大风子只作外用，祛风燥湿、攻毒杀虫，治麻风、梅毒、疥癣。

蛇床子外用燥湿杀虫止痒，治阴部湿痒、湿疹、湿疮、疥癣；内服温肾散寒除湿，治阳痿、宫冷不孕、带下、腰痛。

木槿皮清热、燥湿、杀虫、止痒，外用治疥癣；内服治带下、泻痢。

丝瓜多以丝瓜络入药，祛风通络，解毒化痰，以通络见长，治风湿痹痛、筋脉拘挛、乳汁不通。

孩儿茶以外用为主，收湿敛疮、生肌止血，治疮疡多脓或久不敛口，外伤出血。

阿魏消积杀虫，治癥瘕痞块、虫积、肉积、心腹冷痛、疟疾、痢疾。

露蜂房祛风除痹、消肿止痛，治痈疽、瘰疬、疮癣、皮肤瘙痒、风虫牙痛，内服、外用均可。
